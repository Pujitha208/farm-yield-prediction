# 🌾 Smart Farming Crop Yield Prediction

## 📌 Project Overview

The **Smart Farming Crop Yield Prediction** project is a Machine Learning-based web application that predicts crop yield based on various agricultural and environmental factors.

This project helps farmers and agricultural analysts make data-driven decisions to improve productivity.

---

## 🚀 Technologies Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Joblib
* Streamlit (Web App)

---

## 📊 Machine Learning Model

* Algorithm Used: Linear Regression
* Problem Type: Regression
* Evaluation Metrics:

  * Mean Absolute Error (MAE)
  * Mean Squared Error (MSE)
  * R² Score

## ⚙️ How to Run the Project

### 1️⃣ Clone the Repository

```bash
git clone <your-repository-link>
cd Farm_Yield_Prediction
```

### 2️⃣ Install Required Libraries

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the Streamlit App

```bash
streamlit run app.py
```

The app will open automatically in your browser.
